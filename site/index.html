How to Install

Step 1: Install Certificate

Open the Certificate Manager (certmgr.msc) and import the provided .pfx file.

Step 2: Add to Trusted Root

Import the certificate into the Trusted Root Certification Authorities store.

Step 3: Enable Test Signing

Run bcdedit /set testsigning on in an elevated Command Prompt and reboot your system.
